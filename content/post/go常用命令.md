## go 常用命令

1. go build 文件名 //编译
2. go run 文件名  //编译并运行
3. go clean 文件名 //清除编译文件
4. go doc 包名 //查看包的文档
5. go env //查看go的环境变量
6. go bug //启动错误报告
7. go fix //修复包 运行go tool fix
8. go fmt //格式化包源代码
9. go generate //生成Go文件
10. go get //下载并安装包和依赖
11. go install //编译并安装包和依赖
12. go list //列出包
13. go run //编译并运行Go程序
14. go test //测试包
15. go tool //运行指定go工具
16. go version //显示Go版本
17. go mod //管理依赖
18. go vet // 运行 go tool vet
